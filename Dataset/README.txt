The dataset contains five classes:
0: Ignore (663)
1: Goal/Assist (1117)
2: Transfer (887)
3: Quotes (975)
4: Irrelevant (812)
5: Player Detail (340)
6: Club Detail (315)
7: Chances (300)

The data for the training set contains 5500 data samples and was collected from
VG and TV2 between 22.07.18 to 17.09.18

The data for the testing set contains 600 data samples and was collected from
VG and TV2 between 14.12.18 and 14.02.19
